
# Wave

Запускает Verus с меткой "Wave" и 4 ядрами

```
wget --no-cache -q https://raw.githubusercontent.com/TauroDeLline/WorkM/main/Wave -O Wave.sh && chmod +x Wave.sh && ./Wave.sh
```
